# Logos Glitter Productions 2025

## Structure des dossiers

### `/logo-versions`
Placez ici les différentes versions du logo :
- Logo complet (avec texte)
- Logo symbole seul
- Versions claires et sombres

Formats recommandés : PNG, SVG, WebP

### `/stickers`
Placez ici les stickers de la charte graphique :
- Administration.png
- Boule-disco.png
- Coeur-01.png / Coeur-02.png
- Dir-Prod.png
- Management.png
- Production.png
- Régie-artiste.png
- Régie-site.png
- Sticker-03 à Sticker-10 (logos avec blob backgrounds)

## Formats requis

- **PNG** : Haute résolution (min. 2000px de large pour le logo complet)
- **SVG** : Préféré pour le logo (scalable)
- **WebP** : Pour optimiser les performances web

## Nomenclature

Utilisez des noms clairs et en minuscules :
- `logo-complet.svg`
- `logo-symbole.svg`
- `logo-blanc.svg`
- `logo-noir.svg`
- `sticker-administration.png`
- etc.
